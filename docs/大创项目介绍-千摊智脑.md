# 千摊智脑 —— 面向小微商户的环境感知型AI经营智能体与数字孪生决策系统

## 一、项目背景与痛点分析

### 1.1 行业现状

中国有超过 **2000 万小微摊贩与个体工商户**，他们是城市"毛细血管"般的经济单元，却长期处于数字化转型的盲区。菜市场摊主、社区便利店、夜市摊位等小微经营者面临的核心痛点包括：

- **记账靠脑子、算账靠本子**：手工记账易出错、难追溯，月底"赚多少不知道"是常态；
- **采购靠经验、浪费很常见**：进多少货全凭感觉，剩菜烂菜损耗率高达 15%-30%；
- **科技产品用不起、不会用**：现有进销存系统面向中型商超，操作复杂、费用高昂，对摊主不友好；
- **食品安全追溯形同虚设**：消费者扫码查不到来源，出现问题时无法精准召回；
- **经营决策全靠"拍脑袋"**：没有数据支撑，不知道明天该进什么、进多少。

### 1.2 核心矛盾

> **小微商户对数字化管理有真实刚需，但市场上缺乏一款"零学习成本、低成本部署、真正懂菜市场"的智能经营工具。**

---

## 二、项目概述

**千摊智脑**是一款专为小微摊贩打造的环境感知型 AI 经营智能体，以**微信小程序为交互入口**、**FastAPI 为云端大脑**、**树莓派 5 + YOLOv8 为边缘感知终端**，构建"语音记账 → 库存管理 → POS 收银 → AI 参谋 → 食品安全追溯"五位一体的智能经营闭环。

项目名称"千摊"寓意服务于千家万户的小摊贩，"智脑"体现其基于人工智能的主动决策辅助能力。

## 三、技术创新点

### 3.1 语音语义融合记账（自然交互）

摊主只需对着手机说"今天进了白菜 50 斤，三毛一斤"，系统自动完成：

- **语音识别**（讯飞 ASR）→ **语义解析**（自研 NLP 规则引擎）→ **商品匹配**（别名模糊映射）→ **批次自动创建**（按保质期规则推算过期日）→ **库存实时更新**

这是国内首个将语音记账与批次级库存管理打通的小微经营系统，将记账操作从"打开APP→选商品→填数量→填价格→保存"的 5 步手动操作，简化为 **1 句自然语言**。

### 3.2 YOLOv8 边缘计算机视觉识别（无感入库）

在摊位部署 **树莓派 5 + CSI 摄像头**，运行 **YOLOv8-nano ONNX** 推理模型：

- 识别 **15 类菜市场常见商品**（白菜、土豆、番茄、黄瓜、西瓜等）；
- 结合 **HX711 称重传感器**，实现"放上去→拍照→识别→称重→入库"全自动；
- **三级降级策略**：CSI 摄像头 → USB 摄像头 → 演示模式，适配不同硬件条件；
- **离线容错**：本地 SQLite 缓存，断网时队列积压，联网后自动幂等同步。

这使得小微摊贩首次能以百元级硬件成本（树莓派约 ¥300）获得计算机视觉入库能力。

### 3.3 环境感知型 AI 经营参谋（数据驱动决策）

系统实时接入 **和风天气 API**（温度、降雨、风速、节气），结合 **Meta Prophet 时序预测模型**与**自研规则引擎**进行集成预测：

- **每日经营建议**：基于库存状态 + 天气影响系数 + 历史销量趋势，生成可解释的三行建议（如"明天下雨概率 80%，黄瓜销量通常下降 20%，建议少进 5 斤"）；
- **What-if 模拟**：摊主可调整进货量、售价等参数，系统预测利润变化，支持三方案对比；
- **临期预警 + 清货建议**：批次级生命周期管理，自动推送临期商品，并给出最优清货折扣率；
- **经验云**（差分隐私 ε=1.0）：匿名聚合全平台商户数据，提供跨商户品类基准，同时保护个体隐私。

### 3.4 数字孪生经营镜像（多维可视化）

构建 **四维数字孪生**模型，将摊位的实时经营状态以可视化方式呈现：

| 镜像维度 | 展示内容 | 数据来源 |
|---------|---------|---------|
| **库存镜像** | 品类柱状图 + 批次生命周期热力图 | 语音记账 + 边缘识别 + POS 销售 |
| **经营镜像** | 7 天/30 天销售趋势 + 客单价分布 | 订单流水实时聚合 |
| **风险镜像** | 六维雷达图（库存积压/损耗/赊账/临期/现金流/价格波动） | 多源融合风险评估 |
| **仪表盘** | 今日营收/利润/库存总值/毛利/临期数 | 实时 SQL 聚合 |

### 3.5 完整的 POS + 食品安全闭环

- **组合支付**：现金/微信/支付宝/赊账四合一，支持挂单取单；
- **日结对账**：营业结束后一键生成日结单，差异自动追踪；
- **批次追溯**：每批次商品从入库到销售全链条记录，消费者扫码可查看产地、进货日期、检测报告；
- **召回销毁**：发现问题批次一键锁定→召回→销毁，全流程可审计。

### 技术栈选型理由

| 技术 | 选型理由 |
|------|---------|
| **微信小程序原生开发** | 摊主无需安装APP，扫码即用，零部署成本 |
| **FastAPI 0.115.6** | 高性能异步 Python 框架，天然适配小团队快速迭代 |
| **PostgreSQL + SQLAlchemy** | 关系型数据的事务一致性，支撑批次追溯和财务对账 |
| **YOLOv8-nano + ONNX** | 百毫秒级推理速度，树莓派 5 可流畅运行 |
| **Meta Prophet** | Facebook 开源的时序预测库，内置节假日建模，适合零售场景 |
| **树莓派 5 + HX711** | ¥300 级边缘计算终端，百元级称重传感器，总成本 < ¥500 |

## 七、应用场景与社会价值

### 7.1 典型用户画像

- **菜市场蔬菜摊主王阿姨**（45 岁，初中文化）：每天凌晨 3 点进货，晚上 7 点收摊。以前账本记在日历背面，现在对着手机说一句话就记好账，系统告诉她明天该进多少货。
- **社区水果店小张**（28 岁）：同时经营微信群团购和线下门店。千摊智脑帮他统一管理线上线下库存，POS 收银自动扣库存，再也不会出现"卖了才发现没货"的尴尬。

### 7.2 社会价值

1. **助力小微商户数字化转型**：以零成本、零门槛的方式，让两千多万个体摊贩首次获得企业级经营管理能力；
2. **减少食物浪费**：AI 采购建议 + 临期清货策略，预计可降低损耗率 5-10 个百分点；
3. **保障食品安全**：批次追溯体系让每棵白菜都能追溯到源头，提升消费者信任；
4. **缩小数字鸿沟**：语音交互 + 极简 UI，让低学历、高龄群体也能享受 AI 红利；
5. **赋能灵活就业**：为地摊经济、夜市经济提供标准化经营工具，降低创业门槛。
