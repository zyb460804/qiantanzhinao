# 千摊智脑 用户手册

> 面向小微商户(菜市场摊主)的操作指南
> 适用版本: 0.1.0 | 更新日期: 2026-07-11

---

## 一、千摊智脑是什么?

千摊智脑是你的 **AI 经营小助手**,帮你用说话的方式记账,还会根据天气、周末、库存情况提醒你明天该进多少货,避免进多了卖不掉烂掉。

### 能帮你做什么?

| 功能 | 解决的问题 |
|------|-----------|
| 🎤 **说话记账** | 不用打字,说一句"进了白菜50斤"就记好了 |
| 📦 **库存看清** | 随时知道还剩多少货,哪些快过期了 |
| 🤖 **进货建议** | 每天告诉你明天该进多少,为什么 |
| 🔮 **沙盘模拟** | "如果我多进50斤会怎样?"提前算给你看 |
| 📊 **经营看板** | 一眼看清今天赚了多少、风险在哪 |

## 二、快速开始

### 2.1 打开小程序

1. 打开微信,扫描体验版二维码
2. 授权登录(无需注册,微信即可)
3. 进入首页,看到今日经营概览

### 2.2 第一次记账(3 步)

```
第1步: 点底部「🎤 记账」标签
第2步: 按住中间的大绿色按钮,说:
       "今天进了白菜50斤，三毛钱一斤"
第3步: 松开手指,看到识别结果,点「✅ 确认入库」
```

**就这么简单!** 一条进货记录就存好了。

## 三、核心功能详解

### 3.1 🎤 语音记账(最常用)

#### 两种输入方式

| 方式 | 适合场景 | 怎么用 |
|------|---------|--------|
| **按住说话** | 大多数情况 | 按住大圆按钮 → 说话 → 松开 |
| **打字输入** | 环境太吵时 | 点顶部「⌨️ 打字」切换 |

#### 能听懂的话(举例)

**进货**(说"进了/买了/上了"):
```
✅ "进了白菜50斤，三毛钱一斤"
✅ "买了土豆三十斤，一块二一斤"
✅ "上了猪肉15斤，十二块钱一斤"
✅ "拉了西瓜20斤，两块一斤，一共花了40块"
```

**卖货**(说"卖了"):
```
✅ "卖了西瓜20斤，两块钱一斤"
✅ "一共卖了40块"
```

**损耗**(说"坏了/扔了"):
```
✅ "扔了烂白菜3斤"
✅ "坏了豆腐2斤"
```

#### 看不懂怎么办?

如果识别结果里有 **黄色标记**,说明系统不太确定。你可以:
- 点「✏️ 修正」修改商品名
- 或者点「✅ 确认入库」如果内容是对的

> **小提示**: 上滑手指可以取消录音重新说。

---

### 3.2 📦 库存查看

点底部「📊 库存」标签:

- **顶部图表**:各品类还剩多少,一目了然
- **库存条**:绿色条越长,库存越多
- **临期提醒**:红色标记的商品要赶紧卖

#### 库存条颜色含义

| 颜色 | 含义 | 建议 |
|------|------|------|
| 长绿条 | 库存充足 | 暂不用进货 |
| 短绿条 | 库存偏少 | 考虑补货 |
| 红色 | 即将断货 | 尽快进货 |

---

### 3.3 🤖 AI 参谋(进货建议)

点底部「🤖 AI参谋」标签,每天会看到:

#### 建议卡片(三行式)

```
┌─────────────────────────────────┐
│ 白菜                    78%置信 │
│                                 │
│ 建议明日采购白菜22斤             │
│                                 │
│ 📋 分析依据                     │
│   近7日平均销量     18斤    (+) │
│   周末客流         预计增加12% (+) │
│   当前库存         不足1天   (+) │
│                                 │
│ ⚠️ 若明日降雨概率超过50%，        │
│    建议减少至15斤               │
└─────────────────────────────────┘
```

**怎么看?**
- `+` 号 = 这个因素支持你多进货
- `-` 号 = 这个因素提醒你少进货
- `⚠️` = 要注意的风险

#### 决策沙盘(试试看)

想模拟"如果我多进点会怎样"?

1. 点「试试:如果我多进50斤会怎样?」
2. 设置进货量、进价、售价
3. 点「运行模拟」
4. 看结果:
   - **净利润**多少
   - **损耗率**高不高(超过30%要小心)
   - AI 给出的建议

---

### 3.4 📊 经营看板

点底部或首页进入「经营看板」:

#### 四个 KPI 数字

| 指标 | 含义 |
|------|------|
| 今日收入(绿) | 今天卖了多少 |
| 今日支出(红) | 今天进货花了多少 |
| 毛利(绿/红) | 收入 - 支出 |
| 风险评分 | 越低越好 |

#### 三个 Tab 切换

| Tab | 看什么 |
|-----|--------|
| 📦 库存 | 各品类库存柱状图 |
| 📈 经营 | 7 天收入和利润走势 |
| ⚠️ 风险 | 六维度风险雷达图 |

#### 风险雷达怎么读?

雷达图六个角代表六种风险,红色区域越大越危险:
- **损耗风险高**:有商品快坏了,赶紧卖
- **天气风险高**:明天可能下雨,少进货
- **库存风险高**:积压太多,要消化

---

### 3.5 🔮 决策沙盘(独立页)

更详细的模拟工具,支持三种方案对比(保守/基准/激进):

1. 选商品
2. 拖动滑块设进货量
3. 填进价和售价
4. 看柱状图对比三种方案的净收益和损耗率

## 四、常见问题

### Q1: 说话听不懂怎么办?

**可能原因 & 解决**:
- 环境太吵 → 换打字输入,或靠近手机说话
- 方言重 → 尽量用普通话,关键词说清楚("进了""白菜""50斤")
- 商品名不对 → 确认商品名在系统支持列表内(白菜/土豆/黄瓜/番茄/苹果/西瓜/豆腐/猪肉等 15 类)

### Q2: 识别结果错了怎么改?

点「✏️ 修正」→ 输入正确的商品名 → 系统自动重新计算 → 确认入库。

### Q3: 建议数量不准?

系统在学习中。你可以:
- 每次按建议进货后,回来点「采纳」反馈
- 系统会越来越懂你的经营习惯

### Q4: 库存数据不对?

检查是否有漏记的进货或销售。补录方法:
- 进货:说"进了XX多少斤"
- 卖货:说"卖了XX多少斤"
- 报损:说"扔了XX多少斤"

### Q5: 需要联网吗?

记账、建议、看板需要联网(2G/3G/4G/WiFi 均可)。树莓派设备支持离线记账,网络恢复后自动同步。

## 五、进阶用法

在「我的」页面可以设置:

## 六、隐私与安全

- 你的经营数据 **只你自己可见**
- 经验云的"同行对比"数据 **完全匿名聚合**(至少 3 家商户才显示,不暴露任何个体)
- 语音录音处理完即删除,不长期保存
