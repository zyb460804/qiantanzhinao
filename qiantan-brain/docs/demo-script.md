# 答辩演示脚本 (Demo Script)

目标：5 分钟内讲清「真实可运行 + 环境感知 + 弱网边缘」三条主线。
⚠️ 标注 **[演示模式]** 的部分为模拟数据，答辩时应如实说明。

## 时间分配

| 时段 | 内容 | 约时长 |
|------|------|--------|
| 0:00–0:40 | 开场 + 业务闭环（语音记账） | 40s |
| 0:40–1:30 | 库存 / 批次 / 采购 / 盘点 | 50s |
| 1:30–2:40 | 数字孪生（库存 / 经营 / 风险） | 70s |
| 2:40–3:20 | What-if 沙盘 | 40s |
| 3:20–4:00 | 环境感知（天气 / 经验云） | 40s |
| 4:00–4:40 | 视觉识别 [演示模式] + 边缘端 | 40s |
| 4:40–5:00 | 总结 + 差异化 | 20s |

## 逐屏话术

### 1. 登录 → 首页经营台
「这是商户首页，今日营收、毛利、库存一目了然。核心数据口径是**真实销售毛利 = 销售收入 − 已售批次成本（FIFO）**，不是简单的收支相减。」

### 2. 语音记账（真实链路）
- 点「语音」→ 录音 → **上传真实音频到后端** → 讯飞转写（需配置 `ASR_*` 凭证）→ 语义解析出商品/数量/金额。
- 展示「确认 / 纠错 / 作废」。
- 若无凭证：明确说明走 **[演示模式]** 随机文本，但不默认开启。

### 3. 库存 / 批次 / 采购 / 盘点
「商品按批次管理，FIFO 扣减、临期预警；采购建议一键生成采购单；盘点支持实盘差异核对与历史。」

### 4. 数字孪生（重点）
- **库存**：柱状图 + **生命周期热力图**（红=临期 / 黄=关注 / 绿=安全 / 灰=无库存）。
- **经营**：近7日 / **近30日切换**，含收入、毛利、**客单价**曲线。
- **风险**：六维雷达（库存 / 天气 / 损耗 / 客流 / **资金** / **品类集中度**）——六维均为**真实计算**。

### 5. What-if 沙盘
「调进货量、价格、天气，实时看收入/利润/风险变化——帮商户做决策。」

### 6. 环境感知
- 天气：配置和风 Key 后返回真实数据 + `source` 标记；无 Key 时如实标注 **[Mock]**。
- 经验云：跨商户匿名聚合（≥3 家才发布，已加 **Laplace 差分隐私噪声**）。

### 7. 视觉识别 [演示模式] + 边缘端
「树莓派端：摄像头识别 + HX711 称重 + **弱网离线缓存**，恢复联网自动同步。
当前 **[演示模式]**：`edge/vision/inference.py` 推理管线已接，但权重为占位；真实权重训练完成后即切换真实识别。」

### 8. 收尾
「差异化：环境感知的需求预测 + 真实毛利口径 + 弱网边缘闭环，而非普通记账工具。」

## 备用
- 准备**离线录屏**（防止现场网络/真机不可用）。
- 真机演示前先跑通 `edge/main.py` 模拟模式与后端 `/edge/ingest` 联调。
